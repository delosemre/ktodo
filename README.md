[EN] It will help you to download Kali Linux tools on Debian/Ubuntu based systems.

# Kurulum/Setup

    git clone https://github.com/delosemre/ktodo
    cd ktodo
    pip install -r requirements.txt
    sudo python3 ktodo.py
